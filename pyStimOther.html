static send_trigger()[source]¶

Triggers recording device by sending short voltage spike from LabJack U3-HV. Spike lasts approximately 0.4 ms if connected via high speed USB (2.0). Ensure high enough sampling rate to reliably detect triggers. Set to use flexible IO #4 (FIO4).
